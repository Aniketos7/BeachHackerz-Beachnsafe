###################
 C'est quoi Beach'n Safe? ###################


Beach'n Safe est une application disponible sur mobile qui à partir d'un service Web développé, 
permet de renseigner un certain nombre d'informations sur les différentes plages du monde. 
Son but principal est d'informer les différents utilisateurs sur les différents risques auxquels ils s'exposent.
De façon spécifique, l'application recense les différentes caractéristiques et données météorologiques de chaque
plage et les analyse de façon intelligente, en se basant sur un algorithme rigoureux, afin de proposer un certain
nombre de recommandations par rapport à la plage  sélectionnée.
